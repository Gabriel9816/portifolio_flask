02 EMENTA
Ementa do PPC: Ementa variável, focalizando tópicos relacionados com tecnologias, aplicações, produtos, metodologias e ferramentas de desenvolvimento de software e hardware.

Ementa Proposta para esta edição da Unidade Curricular.
Criar rotas e integrar páginas HTML. Manipular dados com formulários web. Autenticação básica em API. Integração com Banco de Dados MySQL. Deploy de API no Google Cloud Platform.


03 OBJETIVO GERAL DA UNIDADE CURRICULAR
Ao final desta Unidade Curricular espera-se que o estudante seja capaz de desenvolver aplicações web com conteúdo dinâmico usando Python e Flask.


04 OBJETIVOS ESPECÍFICOS DA UNIDADE CURRICULAR
- Demonstrar o uso adequado dos conceitos para desenvolvimento de páginas web com conteúdo dinâmico.
- Demonstrar o uso adequado da linguagem de programação Python.
- Demonstrar o uso adequado do micro-framework Flask.

Observações/Detalhamento das Avaliações
Nota 1 - Portifólio de atividades feitas em sala: consiste da entrega de listas de exercícios práticos realizados durante as aulas do semestre. Esta apresentação acontecerá a partir de uma página web dinâmica, a qual deverá ler as respostas das atividades armazenadas em um banco de dados relacional, e exibí-las na página (portifolio) de cada estudante.
Nota 2 - Projeto Prático: consiste em modelar e desenvolver uma solução web básica completa durante o semestre letivo. Cada componente da página deverá ser desenvolvido a medida que os conteúdos forem apresentados nas aulas.


06 RECUPERAÇÃO DA APRENDIZAGEM
Revisão de conteúdos às vésperas e após a realização dos instrumentos de avaliação. Além de atendimento individualizado para os estudantes interessados durante os horários de Permanência. Recuperação de aprendizagem constante será feita através da implementação ao estilo hands on de alguns algoritmos presentes em listas de Exercícios extra classe. Estas listas de exercícios extras para prática de programação serão disponibilizadas ao longo do semestre, com apresentação da correção das atividades mais estratégicas no decorrer das aulas. Para recuperação de nota será adotado o instrumento de avaliação "Prova Oral", com nota máxima de dez pontos, a qual poderá substituir a menor das notas dos outros instrumentos de avaliação que o estudante tiver apresentado
